# 📊 Sales Analysis – 7 Interactive Dashboards (Dark & Light Mode)

## 📌 Overview  
This project showcases a complete end-to-end Sales Analysis solution developed in Power BI.  
It consists of 7 fully interactive dashboards designed in both Dark & Light modes to enhance user experience and accessibility.

The objective was to combine strong analytical thinking with clean UI/UX design, ensuring smooth navigation, efficient performance tracking, and clear visibility of key business metrics.

The dashboards help stakeholders monitor performance, identify trends, and explore insights dynamically through interactive features.

<hr/>

## 🛠 Tools Used  

- Power BI – Dashboard development and data modeling  
- DAX – KPI calculations and advanced business metrics  
- Power Query – Data cleaning and transformation  
- Figma – Custom dashboard background and UI design  
- Bookmarks & Navigation Features – Interactive report control  
- Drill-through & Drill-down – Detailed performance analysis  

<hr/>

## 🌟 Project Highlights  

- Developed 7 interactive dashboards in both Dark & Light themes  
- Designed custom dashboard backgrounds using Figma for professional UI consistency  
- Built a seamless navigation panel for smooth movement across reports  
- Implemented bookmarks for dynamic filters and customized views  
- Added interactive tooltips to provide contextual insights on demand  
- Created drill-down and drill-through functionality for deeper analysis  
- Developed advanced DAX measures to power KPIs and business metrics  
- Focused on clean layout, structured storytelling, and strong visual hierarchy
